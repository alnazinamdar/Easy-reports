O que é DAO(Data Access Object)?    
    Data Acces Object do inglês em uma tradução livre Objeto de Acesso de Dados.
    Este, é um objecto que utilizamos na programação para abstrair e realizar a comunicação com o Banco de Dados ou sistema de persistência de dados.
    Um exemplo prático do uso do DAO é dado a seguinte situação: Você possui um sistema de cadastro de usuários com todas os seus respectivos atributos.
    Nome, sobrenome, data de nascimento, email, usuario, senha, telefone, entre outros. 
    Nesse sistema o administrador poderá Adicionar novos usuários, remover usuários existentes, atualizar informações de usuários
    e por consequencia buscar os usuários do banco.
